# openfoodfact-api

## Description

L'objectif de cette API NestJS est de permettre d'obtenir des informations sur des codes barres via l'API OpenFoodFact.

## Installation

```bash
$ npm install
```

```
change constants in Auth
Add config file to connect to the bdd
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
